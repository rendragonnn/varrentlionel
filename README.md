[![Live Demo](https://img.shields.io/badge/Live%20Demo-varrentlionel.vercel.app-64ffda?style=for-the-badge&logo=vercel)](https://varrentlionel.vercel.app)

A personal portfolio website for **Varrent Lionel**, an undergraduate Information Systems student specializing in **Big Data Analysis** at Universitas Multimedia Nusantara. The site showcases skills, projects, and experience as a Data Analyst, BI Analyst, and Front End Developer.

---

## 🌐 Live Site

👉 [https://varrentlionel.vercel.app](https://varrentlionel.vercel.app)

---

## ✨ Features

- **Animated hero section** with a typewriter effect cycling through roles
- **Particle background** powered by particles.js
- **Scrolling skills marquee** showcasing the tech stack
- **Projects section** with interactive tilt cards
- **Experience section** with image lightbox support
- **Contact section** for getting in touch
- **Magnetic cursor effects** and smooth scroll-reveal animations
- **Fully responsive** design
- **SEO optimised** with Open Graph and Twitter Card meta tags
- **Accessibility** skip-navigation link and semantic HTML
