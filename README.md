# Varrent Lionel — Portfolio V2

> 🌐 Live Preview: [varrentlionel.site](https://varrentlionel.site)

A personal portfolio built with React and Vite, featuring a technical-brutalist aesthetic tailored for a **Business Intelligence Analyst**. This project showcases my professional experience, data analytics projects, and organizational journey.

---

## 📸 Preview

![Portfolio Animation](public/preview.gif)

## 🚀 Features

- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Premium Aesthetics**: Warm cream theme with technical wireframe animations.
- **Project Showcase**: Asymmetric bento grid for analytical documentation.
- **Interactive Experience**: Differentiated Work and Organizational timeline with soft-skill tags.
- **CV Preview Modal**: View my resume directly without leaving the page.
- **Functional Contact Form**: Direct message integration via Formspree.

## 🛠️ Tech Stack

- **Frontend**: React.js, Vite
- **Styling**: Vanilla CSS, Custom Properties
- **Animations**: CSS Keyframes (GPU Accelerated)
- **Icons**: FontAwesome 6
- **Deployment**: GitHub Pages / Vercel
